# Your First Mutation 🚀

You're ready! Here's how to create your first AI mutation:

## Step 1: Select a Component
Open the Vanguard Preview (Cmd/Ctrl + Shift + V) and click on any component.

## Step 2: Describe the Change
In the "Mutation Input" box (right panel), type what you want:

**Examples:**
- "Add a loading spinner"
- "Make the button red"
- "Add validation for email"
- "Show an error message if count is negative"

## Step 3: Watch the Magic ✨
Hit Enter or click "Mutate". Vanguard will:
1. Analyze your component's props and imports
2. Query DeepSeek for the best mutation code
3. Generate the new code in seconds
4. Show you the result with syntax highlighting
5. Apply it to the file (with a git commit)
